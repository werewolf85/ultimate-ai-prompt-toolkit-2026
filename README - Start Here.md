## How to Use
Each prompt has the format:
**Title**
**Purpose:** What the prompt achieves
**Best Model:** GPT-4o, Claude 3.5, Midjourney, etc.
**Prompt:** The prompt text with [variables]
**Example:** Example usage
**Expected Result:** What you will get as output

Replace [variables] with your data and get results.
