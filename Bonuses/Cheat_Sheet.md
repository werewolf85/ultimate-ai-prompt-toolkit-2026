# Cheat Sheet for Prompt Engineering

## Core Principles
- Clarity: be specific in your request.
- Context: provide necessary information.
- Format: specify desired output format.
- Role: assign the AI a role (e.g., you are an expert in...).
- Examples: use few-shot learning when needed.

## Templates
1. **Closed Question:** Answer [question] with explanation.
2. **Creative:** Create [something] in [style].
3. **Analysis:** Analyze [object] by criteria: [list].
4. **Transformation:** Convert [text] to [format] preserving [elements].
5. **Comparison:** Compare [A] and [B] by [parameters].

## Quick Checks
- [ ] Is the purpose clear?
- [ ] Is there enough context?
- [ ] Is the output format specified?
- [ ] Are examples provided if needed?
- [ ] Is there any ambiguity?
